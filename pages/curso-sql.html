Curso en Vivo

SQL para Data Analyst (Básico - Intermedio)

Temario del Curso

Módulo 1: Introducción a Bases de Datos y Modelamiento

Aprende los fundamentos desde cero, incluyendo la definición de modelos (conceptual, lógico y físico), componentes como entidades y atributos, creación de diagramas ER y la normalización de tablas.

Módulo 2: Lenguaje Transact SQL y Funciones Nativas

Domina el uso de funciones de agregación clave (Count, max, min, sum, avg), además de implementar funciones específicas para el manejo de tipos de datos de fecha, texto y matemáticas.

Módulo 3: Manipulación de Datos y Estructuras de Control

Conoce cómo operar sobre tu base de datos ejecutando consultas condicionales mediante el uso de comandos fundamentales como select, insert, update, delete, from, where, group by y order by.

Módulo 4: Operadores Lógicos y Sentencias Avanzadas

Profundiza en la lógica de las consultas aplicando operadores lógicos (and, or, not, in, like, between) y domina el uso avanzado de sentencias como CAST, CASE WHEN y Group By.

Módulo 5: Relaciones y Consulta de Múltiples Tablas (JOINS)

Aprende a conectar y cruzar información entre diferentes tablas dominando todos los tipos de uniones: Inner Join, Outer Join, Cross Join y Left Join.

Módulo 6: Subconsultas y Funciones de Ventana

Da el salto al análisis avanzado implementando Common Table Expressions (CTEs) y comparando su uso frente a las Window Functions, incluyendo funciones de Ranking y agregativas.

Módulo 7: Resolución de Casos Prácticos y Prueba Técnica

Pon a prueba lo aprendido a través de una sesión de repaso intensiva y el desarrollo de una prueba técnica real diseñada para evaluar tus habilidades analíticas.

Módulo 8: Automatización de Reportes (Bonus Track)

Aprende las mejores prácticas para automatizar la ejecución y entrega de tus reportes en SQL, integrando soluciones de forma nativa o mediante Python. (Bonus Track)
