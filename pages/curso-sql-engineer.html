Curso en Vivo

SQL para Data Engineer (Intermedio - Avanzado)

Temario del Curso

Módulo 1: Fundamentos Relacionales y Consultas Clave

Refuerza tus bases en bases de datos relacionales, dominando consultas esenciales, operadores y control condicional preparatorio para entornos de alta exigencia.

Módulo 2: Dominio de Subconsultas, CTEs y Window Functions

Analiza en profundidad el uso de subconsultas correlacionadas vs. no correlacionadas, y domina la implementación avanzada de CTEs y Window Functions.

Módulo 3: Arquitectura y Gestión en Google BigQuery

Aprende a crear y gestionar datasets, configurar tablas externas, y conectar fuentes de datos externas para ejecutar potentes consultas federadas en la nube.

Módulo 4: Optimización de Consultas y Reducción de Costos

Comprende el motor de BigQuery a fondo para aplicar particionamiento y clustering estratégico, optimizando el rendimiento y reduciendo los tiempos y costos de ejecución.

Módulo 5: Funciones SQL Avanzadas y Datos Semiestructurados

Aprende a manipular arrays y structs, limpiando texto con expresiones regulares y trabajando directamente con datos en formato JSON desde SQL.

Módulo 6: Funciones Estadísticas y UDFs Personalizadas

Amplía tu capacidad analítica en BigQuery utilizando funciones estadísticas y aproximadas, y aprende a crear y desplegar tus propias funciones personalizadas (UDFs).

Módulo 7: Subconsultas Avanzadas y Vistas Materializadas

Combina técnicas complejas de SQL con las optimizaciones de BigQuery, creando vistas materializadas que te permitirán reutilizar lógica de negocio de manera eficiente.

Módulo 8: Proyecto Integrador: Construcción de Pipeline ELT

Construye un mini pipeline ELT desde cero: ingesta datos crudos, límpialos utilizando CTEs y genera una vista materializada de métricas de negocio para tu prueba técnica final. (Bonus: Desarrollo de prueba técnica)
